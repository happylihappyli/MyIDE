
现在还有哪些AI IDE里有的，我这里没有


- 自动提取报错回传 ：当编译或运行失败时，AI IDE 可以自动抓取终端的错误信息并直接发给 AI 提问。我们现在还需要点击那个“📋 复制给 AI”按钮手动带回去。
- Inline Diff（行内差异预览） ：目前的 MyIDE 在“预览”时，是在左侧粗略显示修改日志。成熟的 IDE 会像 Git 那样，在代码编辑框里用红色和绿色 直观地对比修改前和修改后的代码 ，并允许您针对某一行点“接受”或“拒绝”。
- 基于 LSP 的语法诊断 ：没有像 VS Code 那样自动标注红色波浪线和代码自动补全。
- 多文件跨光标引用 ：缺少自动追踪您当前正在阅读的代码符号跳转等上下文隐式感知能力。
